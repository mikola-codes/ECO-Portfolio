# Team ECOZONE — OJT Portfolio Website

> Professional web portfolio of the IT Officer Trainees from Western Mindanao State University (WMSU) — College of Computing Studies, assigned at the Zamboanga City Special Economic Zone Authority & Freeport (ZAMBOECOZONE) Management Information Systems Division.

---

## About The Project

This repository contains the source code for the official online internship portfolio of Team ECOZONE. The website documents our On-the-Job Training (OJT), covering hands-on IT operations, system development, and networking tasks.

The core highlight of our internship and this portfolio is the development of the Biometric Fingerprint Attendance System.

## Features

- **Corporate UI/UX:** Styled using ZAMBOECOZONE's official color palette.
- **Typing Text Animation:** Dynamic hero section using pure JavaScript.
- **Animated Stats:** Interactive visual counters.
- **Interactive Timeline:** A detailed record of tasks and milestones.
- **OJT Gallery & System Interface:** Image galleries showcasing real-world moments and the system UI.
- **Animated Skill Bars:** Visual representation of technical proficiencies.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop viewing.
- **AOS Scroll Animations:** Smooth fade and zoom effects on scroll.

## How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/ecozone-portfolio.git
   ```
2. **Open the project folder:**
   Navigate into the downloaded folder.
3. **Launch the site:**
   - Open `index.html` using VS Code Live Server.
   - Alternatively, double-click `index.html` to open it directly in any modern web browser.

> Note: An active internet connection is required to load Google Fonts and the AOS animation library.

## Deploying on GitHub Pages

1. Push this project to your GitHub repository.
2. Go to your repository's Settings → Pages.
3. Under Build and deployment, set the source branch to main (or master) and folder to / (root).
4. Click Save. Your site will be live at: https://yourusername.github.io/repo-name/
